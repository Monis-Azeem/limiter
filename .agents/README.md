# Project Agent Docs

This folder stores planning and engineering guardrails for the mobile focus-control app.

- `product-roadmap.md`: product scope, V1 definition, milestones, and deployment path.
- `architecture-and-folders.md`: monorepo structure and module boundaries.
- `ui-style-guide.md`: clean ChatGPT-like UI system for mobile.
- `engineering-standards.md`: coding, testing, release, and quality rules.
- `skills-options-from-skills-sh.md`: candidate installable skills discovered via `npx skills find`.
- `v1-execution-decisions.md`: locked choices based on free distribution and hard-enforcement constraints.
- `mobile-enforcement-architecture.SKILL.md`: custom skill for hard-enforcement architecture and anti-bypass patterns.
- `rn-clean-minimal-ui.SKILL.md`: custom skill for ChatGPT-like clean UI patterns in RN.
- `offline-release-distribution.SKILL.md`: custom skill for free-first distribution planning.
- `mobile-quality-gates.SKILL.md`: custom skill for test/release quality gates.
