# Distribution Plan

## Android (Free-First)

- Generate signed APK artifacts.
- Share directly via file or secure link.
- Include in-app version display and update prompt.

## iOS (Free Constraint)

- Personal Team sideload for very limited internal testers only.
- Not suitable for broad distribution.

## iOS Scalable Path

- Enroll in Apple Developer Program.
- Shift to TestFlight for broader testing.
- Keep entitlement and policy docs updated for Family Controls usage.
